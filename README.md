# commercial_analysis

## 예비 창업자를 위한 창업 위치 추천 프로그램

### 예비 창업자가 창업하기 희망하는 업종과 지역을 입력하면, 이미 창업 되어있는 업종 중 가장 인기있는 top10 업체의 주변 지역 환경을 분석해서, 공통된 지역 환경을 가지지만 아직 해당 업체가 들어서지 않은 지역을 찾아서 추천해주는 프로그램

### 프로젝트 소개 : ./최종 제출 자료/시크릿_스페이스_결과보고서_최종.hwp 참고
